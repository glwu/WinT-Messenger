WinT Messenger Documentation

Introduction

WinT Messenger is a practical instant messaging application that its developed by a small team of students working in the WinT 3794 team. The application is written in QML/C++ and can be used with the most popular desktop and mobile operating systems. See the features page for more information.

You can see how the whole app works here, or even write code to make it better!
